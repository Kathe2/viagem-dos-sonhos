# ✈️ Travel Agency Website

Projeto de site para agência de viagens desenvolvido com **HTML, CSS e JavaScript** durante o módulo de **Front-End** da *Faculdade Descomplica*.

O projeto teve como objetivo praticar a construção de interfaces modernas e responsivas, aplicando conceitos fundamentais de desenvolvimento front-end e criação de páginas web voltadas para experiência do usuário.

---

## 📌 Sobre o projeto

A aplicação simula o site de uma agência de viagens, apresentando destinos turísticos, pacotes de viagem e informações organizadas em uma interface visual moderna e responsiva.

Durante o desenvolvimento, foram trabalhados conceitos importantes de estruturação de páginas, estilização e interatividade utilizando JavaScript puro.

---

## 🚀 Funcionalidades

✔️ Página inicial moderna e responsiva

✔️ Seção de destinos turísticos

✔️ Exibição de pacotes de viagem

✔️ Navegação entre seções da página

✔️ Layout adaptável para diferentes dispositivos

✔️ Elementos interativos com JavaScript

✔️ Design focado em experiência do usuário

---

## 🛠️ Tecnologias utilizadas
